Commandes Git essentielles

Tableau récapitulatif des commandes Git courantes.

Commande Description
git init Crée un nouveau dépôt Git local dans le répertoire courant.
git clone <url> Copie un dépôt distant en local (récupère l'historique et les fichiers).
git add <fichier(s)> Ajoute des fichiers à l'index (prépare pour le commit).
git commit -m "message" Enregistre les modifications indexées dans l'historique avec un message.
git status Affiche l'état du dépôt : fichiers modifiés, indexés, non suivis, etc.
git push Envoie les commits locaux vers le dépôt distant configuré (remote).
