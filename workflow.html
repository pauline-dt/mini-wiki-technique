Explication du workflow Git simple

Feature branches → merge → PR

Étapes

  1. Créer une branche de fonctionnalité
  2. Travailler et committer localement
  3. Pousser la branche sur le dépôt distant
  4. Ouvrir une Pull Request (PR) pour revue
  5. Après validation, merger la PR dans la branche cible (ex. main)
