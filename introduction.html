Introduction à Git

Pourquoi utilise-t-on Git ?

Git est un outil super pratique pour suivre l’évolution d’un projet. Au lieu d’écraser les fichiers des autres ou de perdre une version importante, Git garde une trace de chaque modification. Ça permet de travailler à plusieurs en même temps sans se marcher dessus, et de revenir en arrière facilement si on s’est trompé. C’est pour ça que quasiment tous les développeurs l’utilisent aujourd’hui : c’est fiable, organisé et ça évite beaucoup de problèmes.

Installation de Git

  1. Télécharger Git depuis le site officiel : https://git-scm.com
  2. Installer la version correspondant à votre système (Windows, macOS ou Linux).
  3. Vérifier l’installation en tapant la commande git --version dans un terminal.
